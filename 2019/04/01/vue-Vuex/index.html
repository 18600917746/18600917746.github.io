解析 Vuex 的四大法宝

第一 state

  • state 是状态对象，对于公用数据的管理
  • mapState 辅助函数
    • 定义在 computed 中使用
    • 映射 this.const = store.state.const

第二 getter store 的计算属性

  • mapGetter 辅助函数
    • 定义在 computed 中使用
    • 数据会进行缓存
    • 数据读取前进行处理计算

第三 Mutation 同步触发转态

  • Mutation 是状态的触发者
    • 定义在 methods 中使用
    • Mutation 必须是同步函数
    • 参数一般使用 解构赋值
  • commit 参数一般传递一个 对象
  • commit 可以使用 type 来指定触发的 函数
  • mapActions 辅助函数

第四 Action 异步触发转态

  • Action 提交的是 mutation，而不是直接变更状态。
    • 定义在 methods 中使用
    • Action 可以包含任意 异步操作
    • 参数一般使用 解构赋值
  • dispatch 参数一般传递一个 对象
  • dispatch 可以使用 type 来指定触发的 函数
  • mapActions 辅助函数

测试环境搭建之 Bugfree 部署

测试环境是什么

  • 测试环境
    • 测试环境 = 软件环境 + 硬件环境 + 网络环境 + 测试数据 + 测试工具
    • 真实、干净、无毒、 独立
    • 模拟真实用户真实的使用场景
  • 软件环境：web服务器、操作系统、数据库管理系统、其他的应用软件
  • 硬件环境：服务器、客户端、网络连接设备等

查看系统的位数

bugfree 管理系统的创建

  1. 使用 xshell 连接远程端
  2. 使用 xftp 上传你下载的 xampp.run 执行文件
  3. 使用 chmod 777 -R xampp.run 来修改文件的操作权限
  4. 执行文件进行文件下载 ./xampp.run
  5. 在 lampp/htdoc 下创建 BugFlie 文件夹
  6. 将 bugfree 文件上传至 lampp/htdoc 文件夹下
  7. 启动服务 ./xampp start
  8. 通过 ip 地址进项访问 xxx.xxx.xx.xx/bugfree

禅道 管理系统的创建

  1. 使用 xshell 连接远程端
  2. 直接将 禅道安装包 放置 /opt 然后进行解压
  3. 直接进行 zbox start
  4. 通过 ip 地址进项访问 xxx.xxx.xx.xx
